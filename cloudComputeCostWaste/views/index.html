Welcome to Cloud Monitoring

The goal of this project is to monitor any instance of a Virtual Machine in a cloud deployment without regard for where that Virtual Machine is in the cloud or even in a local network. As long as that Virtual Machine has network access to this host it can relay probe data. Data is logged and displayed to the application own with the intention of displaying cost and in particular waste in that Virtual Machine instance.
